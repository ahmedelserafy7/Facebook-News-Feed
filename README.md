This project is a mocking app like **Facebook News Feed**.

## Used Skills

1. Creating **Tab Bar** menu.
2. Using **Dynamic Cell Sizing**.
3. Implementing **Image Zoom**.
4. Storing Images in **Cache**.

 
## Requirements

This project requires: 
* **Xcode 10+** 
* **iOS 12.0+**
